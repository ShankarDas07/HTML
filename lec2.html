TABLE

TABLE : An HTML table is a structured way to display data in rows and columns on a web page. It allows authors to organize information such as text, numbers, images, or even other tables, into a grid-like format for clear presentation.

Key elements of an HTML table :

  • < table > : The main container element that defines the entire table.
  • < tr > : Defines a table row, containing one or more cells.
  • < th > : Defines a table header cell, typically displayed in bold and used for column headings.
  • < td > : Defines a standard table data cell, containing the actual data.
  • < thead > : Groups the header content of the table.
  • < tbody > : Groups the main body content of the table.
  • < tfoot > : Groups the footer content of the table.
  • < caption > : Provides a short description or title for the table.
