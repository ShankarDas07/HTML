What is Web Development

Web Development is the process of building and maintaining websites or web applications that run on the internet.
Examples: Facebook, Amazon, Netflix, Gmail.

Web development generally has three major parts:

  1. Frontend Development (Client Side)

    This is the part of a website that users see and interact with in their browser.

    Key Points:
    • Handles design, layout, and interactivity.
    • Languages/Technologies:
      • HTML → Provides the structure of the webpage (skeleton).
      • CSS → Adds design, colors, spacing, and layout.
      • JavaScript → Makes the page interactive (buttons, animations, dynamic content).
  2. Backend Development (Server Side)

    This is the part that users cannot see. It handles the logic, database, and server operations.

    Key Points:
    • Processes user requests (like login, payments, data storage).
    • Works with databases to store or retrieve information.
    • Languages/Technologies:
      • Java (with Spring Boot), Node.js, Python, PHP, etc.
      • Databases: MySQL, MongoDB, PostgreSQL, etc.
  3. Full-Stack Development

    A Full-Stack Developer works on both frontend and backend, building the entire application from the user interface to the database.

    1. Java Development (in a Web Development Course)

      If your course teaches Java Development, you will typically learn:

      • Core Java: Basics of Java programming (OOP, loops, arrays, classes).
      • Advanced Java: Servlets, JSP (Java Server Pages), JDBC (database connection).
      • Frameworks:
        • Spring Boot → A powerful Java framework to create modern web applications quickly.
      • Database integration with MySQL or PostgreSQL.
      • Building full web applications using Java on the backend.
    2. MERN Stack

      MERN is a JavaScript-based Full-Stack Development technology set.

      It stands for:

      • M → MongoDB (Database)
      • E → Express.js (Backend web framework)
      • R → React.js (Frontend library)
      • N → Node.js (Server runtime for JavaScript)

      With MERN you can build complete web apps using only JavaScript for both frontend and backend.
